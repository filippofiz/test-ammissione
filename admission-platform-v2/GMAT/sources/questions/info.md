# Id Coding Conventions

"XX-CCCC-SSSS-00000"
XX is the subject area code so:
- QR = Quantitative Reasoning
- VR = Verbal Reasoning
- DI = Data Insights

First CCCC is a code that indicates the source of the question:
- GMAT = Official GMAT question
- AI__ = AI-generated question
- OT__ = Other source question

Second SSSS is a code that is present only for Official GMAT questions to indicate the source:
- SK__ = GMAT Starter Kit
- SI__ = GMAT Skill Insight

00000 is a five digit number that is unique to each question.
