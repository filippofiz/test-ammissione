# Score Cycles & Progression Criteria

## Cycle Definitions

### Cycle 1: Beginner
- **Target Score Range**: 505-555
- **Typical Starting Profile**: Limited GMAT familiarity, foundational gaps
- **Difficulty Distribution**: 70% Easy, 25% Medium, 5% Hard
- **Focus Areas**: Building fundamentals, time management basics, question type familiarity

### Cycle 2: Intermediate
- **Target Score Range**: 555-615
- **Typical Starting Profile**: Some preparation, inconsistent performance
- **Difficulty Distribution**: 30% Easy, 50% Medium, 20% Hard
- **Focus Areas**: Strengthening weak areas, developing strategies, improving accuracy

### Cycle 3: Advanced
- **Target Score Range**: 615-675
- **Typical Starting Profile**: Solid foundations, aiming for competitive scores
- **Difficulty Distribution**: 10% Easy, 40% Medium, 50% Hard
- **Focus Areas**: Advanced strategies, speed optimization, error prevention

### Cycle 4: Elite
- **Target Score Range**: 675-715+
- **Typical Starting Profile**: Strong foundations, targeting top programs
- **Difficulty Distribution**: 5% Easy, 25% Medium, 70% Hard
- **Focus Areas**: Excellence-level strategies, mental resilience, peak performance

## Initial Placement

### Diagnostic Simulation
- Full GMAT format (64 questions, 2h 15min)
- Adaptive algorithm mimics real GMAT
- Completed before any preparation begins

### Placement Criteria

| Diagnostic Score | Starting Cycle |
|-----------------|----------------|
| Below 505 | Cycle 1 (Beginner) |
| 505-550 | Cycle 1 (Beginner) |
| 555-650 | Cycle 2 (Intermediate) |
| 650-705 | Cycle 3 (Advanced) |
| 710-725 | Cycle 4 (Elite) |

### Special Cases
- **Below 505**: Start with Cycle 1, may need extended fundamentals work
- **Above 715**: Already at elite level, focus on specific weak areas and test-taking optimization

## Progression Criteria

Students advance to the next cycle when they meet the following criteria:

### Primary Criteria (must meet at least one)
1. **Section Assessment Performance**: Score consistently in target range on 2+ section assessments
2. **Mock Simulation Performance**: Full mock score reaches cycle target

### Supporting Criteria (considered for borderline cases)
- Topic assessment trends (improving scores across topics)
- Training session performance
- Tutor evaluation of conceptual mastery
- Student confidence and readiness

## Progression Decision Framework

### Ready to Advance
- Meets primary criteria
- Shows consistent improvement
- Demonstrates strategic thinking
- Time management is appropriate for next level

### Not Ready - Continue Current Cycle
- Scores fluctuate significantly
- Fundamental gaps remain
- Time management issues persist
- Additional practice needed on specific topics

### Tutor Discretion
The tutor has final say on progression. Consider:
- Student's target score and timeline
- Quality vs. speed of advancement
- Student's confidence level
- Specific weakness patterns

## Cycle Repetition

If a student needs to repeat topics within a cycle:
- Use different questions from the same difficulty pool
- Focus on specific weak areas identified
- Reduce time on topics already mastered
- Consider targeted mini-assessments

## Timeline Considerations

### Typical Duration Per Cycle
- **Minimum**: 20 weeks (aggressive pace, 3 lessons/week, no repeats)
- **Standard**: 24-30 weeks (2-3 lessons/week)
- **Extended**: 36+ weeks (additional practice, topic repeats)

### Multi-Cycle Students
For students starting at Beginner targeting Elite:
- Plan for 3 full cycles
- Estimate 18-24 months total preparation
- Set intermediate milestones for motivation

## Score Conversion Reference

### GMAT Score Ranges
| Percentile | Total Score | Description |
|------------|-------------|-------------|
| 99th | 755-805 | Exceptional |
| 90th | 700-750 | Excellent |
| 75th | 650-695 | Very Good |
| 50th | 570-645 | Average |
| 25th | 500-565 | Below Average |

### Section Score Ranges
Each section (QR, VR, DI) scores from 60-90:
- 60-65: Needs significant improvement
- 66-72: Below average
- 73-79: Average
- 80-84: Above average
- 85-90: Excellent
