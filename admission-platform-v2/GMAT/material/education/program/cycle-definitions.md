# Score Cycles & Progression Criteria

## Cycle Definitions

The UpToTen GMAT preparation program uses a 3-cycle progressive system. Each cycle has a different number of lessons per topic, reflecting the student's increasing independence as they advance.

### Cycle 1: Foundation
- **Target Score Range**: 505-605
- **Expected Score Improvement**: +100 to +120 points
- **Typical Starting Profile**: Limited GMAT familiarity, foundational gaps, first-time test takers
- **Lessons per Topic**: 4 lessons (A, B, C, D)
- **Lesson Duration**: ~90 minutes each
- **Total Tutor Hours**: ~72 hours (12 topics × 4 lessons × 1.5h)
- **Autonomous Work Hours**: ~30 hours
- **Difficulty Distribution**: 60% Easy, 30% Medium, 10% Hard
- **Focus Areas**: Building fundamentals, time management basics, question type familiarity, concept mastery through repetition

### Cycle 2: Development
- **Target Score Range**: 605-665
- **Expected Score Improvement**: +50 to +60 points
- **Typical Starting Profile**: Some preparation, basic concepts understood, inconsistent performance
- **Lessons per Topic**: 3 lessons (A, B, C)
- **Lesson Duration**: ~90 minutes each
- **Total Tutor Hours**: ~54 hours (12 topics × 3 lessons × 1.5h)
- **Autonomous Work Hours**: ~35 hours
- **Difficulty Distribution**: 25% Easy, 50% Medium, 25% Hard
- **Focus Areas**: Strengthening weak areas, developing strategies, improving accuracy, building speed

### Cycle 3: Excellence
- **Target Score Range**: 665-715+
- **Expected Score Improvement**: +25 points
- **Typical Starting Profile**: Solid foundations, aiming for competitive scores, targeting top programs
- **Lessons per Topic**: 2 lessons (A, B)
- **Lesson Duration**: ~90 minutes each
- **Total Tutor Hours**: ~36 hours (12 topics × 2 lessons × 1.5h)
- **Autonomous Work Hours**: ~40 hours
- **Difficulty Distribution**: 5% Easy, 30% Medium, 65% Hard
- **Focus Areas**: Advanced strategies, speed optimization, error prevention, problem modeling, mental resilience

## Initial Placement

### Two-Step Assessment Process

Students undergo a two-step assessment process before starting their preparation:

#### Step 1: Familiarization Mock Test
- **Purpose**: Introduce students to GMAT question formats
- **Duration**: ~45 minutes
- **Format**: ~20 representative questions covering all sections (QR, DI, VR)
- **Evaluation**: Not scored for placement; purely educational
- **Content**: Examples with explanations of each question type

#### Step 2: Placement Assessment (Simil-GMAT)
- **Purpose**: Determine appropriate starting cycle
- **Duration**: ~1.5 hours
- **Format**: ~45 questions in GMAT-like style (original questions, not official GMAT)
- **Evaluation**: Scored to determine cycle placement
- **Output**: Estimated score range and cycle recommendation

### Placement Criteria

| Assessment Score | Starting Cycle |
|-----------------|----------------|
| Below 505 | Cycle 1 (Foundation) - Extended |
| 505-600 | Cycle 1 (Foundation) |
| 600-660 | Cycle 2 (Development) |
| 660+ | Cycle 3 (Excellence) |

### Special Cases
- **Below 505**: Start with Cycle 1 (Foundation), may need extended fundamentals work with additional pre-cycle preparation
- **Above 700**: Start with Cycle 3 (Excellence), focus on specific weak areas and peak performance optimization

## Progression Criteria

Students advance to the next cycle when they meet the following criteria:

### Primary Criteria (must meet at least one)
1. **Section Assessment Performance**: Score consistently in target range on 2+ section assessments
2. **Mock Simulation Performance**: Full mock score reaches cycle target

### Supporting Criteria (considered for borderline cases)
- Topic assessment trends (improving scores across topics)
- Training session performance
- Tutor evaluation of conceptual mastery
- Student confidence and readiness

## Progression Decision Framework

### Ready to Advance
- Meets primary criteria
- Shows consistent improvement
- Demonstrates strategic thinking
- Time management is appropriate for next level

### Not Ready - Continue Current Cycle
- Scores fluctuate significantly
- Fundamental gaps remain
- Time management issues persist
- Additional practice needed on specific topics

### Tutor Discretion
The tutor has final say on progression. Consider:
- Student's target score and timeline
- Quality vs. speed of advancement
- Student's confidence level
- Specific weakness patterns

## Cycle Repetition

If a student needs to repeat topics within a cycle:
- Use different questions from the same difficulty pool
- Focus on specific weak areas identified
- Reduce time on topics already mastered
- Consider targeted mini-assessments

### Multi-Cycle Students
For students starting at Foundation targeting Excellence:
- Plan for all 3 cycles
- Estimate 10-15 months total preparation
- Set intermediate milestones for motivation
- Total hours: ~102h + ~89h + ~76h = ~267 hours (tutor + autonomous)

## Score Conversion Reference

### GMAT Score Ranges
| Percentile | Total Score | Description |
|------------|-------------|-------------|
| 99th | 755-805 | Exceptional |
| 90th | 700-750 | Excellent |
| 75th | 650-695 | Very Good |
| 50th | 570-645 | Average |
| 25th | 500-565 | Below Average |

### Section Score Ranges
Each section (QR, VR, DI) scores from 60-90:
- 60-65: Needs significant improvement
- 66-72: Below average
- 73-79: Average
- 80-84: Above average
- 85-90: Excellent
