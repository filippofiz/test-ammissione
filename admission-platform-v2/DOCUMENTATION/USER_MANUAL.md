# User Manual

End-user guide for students and tutors.

---

## 👨‍🎓 For Students

### Getting Started
1. **Receive invitation** from your tutor
2. **Click registration link** in email
3. **Create password** (min 6 characters)
4. **Log in** to dashboard

### Taking a Test
1. **Select test** from "My Tests"
2. **Read instructions** carefully
3. **Start test** - timer begins
4. **Answer questions** - select your choice
5. **Submit** - review before final submit

### Tips
- ⏰ Watch the timer
- 📑 GMAT: Can't go back to previous questions
- 📊 SAT: Adaptive - difficulty adjusts based on performance
- 🔖 Bookmark difficult questions (SAT only)

---

## 👨‍🏫 For Tutors

### Student Management
1. **Add students** - Send invitation via email
2. **Assign tests** - Choose tests for each student
3. **Track progress** - View completion status
4. **Review results** - Analyze performance

### AI Question Generation
1. **Go to** "Generate Questions"
2. **Select** question type, difficulty, topic
3. **Generate** - AI creates questions
4. **Review** - Approve or reject
5. **Assign** - Add to tests

### Best Practices
- Review all AI-generated questions before use
- Provide feedback on rejected questions
- Monitor student progress regularly
- Adjust test assignments based on performance
